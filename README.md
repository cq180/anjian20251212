# anjian20251212

`anjian20251212` 是一个基于 Python/Tkinter 的 Windows 桌面自动化工具。它可以把鼠标、键盘、找图、等待、循环、定时和组合执行等操作组织成快捷脚本，适合用来做重复性点击、日常流程整理、游戏养成流程辅助、软件操作自动化和个人效率工具。

项目核心思路是：先把常用操作录制或写成脚本，再绑定到可视化按钮上，需要时一键执行。

## 适用场景

这个工具适合处理规则明确、重复性高、需要人工反复操作的流程，例如：

- 阴阳师等养成类游戏的日常流程辅助：例如固定入口点击、每日任务路线、活动页面切换、御魂/探索/结界等重复操作的快捷执行。
- 养成类、放置类、回合制游戏的快捷脚本：例如定时领取资源、重复点击确认、按照固定顺序切换页面、执行多段组合脚本。
- 抢票或预约类场景的个人辅助：例如提前打开页面、定时触发点击、重复刷新和确认等简单操作。
- 普通桌面软件自动化：例如批量点击、复制粘贴、打开程序、按固定步骤处理文件。
- 脚本学习和调试：通过录制操作生成脚本，再在编辑器里查看、修改和单步执行。

请注意：本项目只提供通用的本地自动化能力。用于游戏、票务、预约或其他第三方平台时，请遵守对应平台规则，不要用于绕过验证码、风控、付费限制、反作弊系统或其他不合规用途。

## 功能概览

- 图形化主控台：提供按钮网格，把脚本绑定到按钮后可以快速启动。
- 单任务区域：适合放常用脚本，左键执行，右键编辑按钮配置。
- 组合执行区域：可以把多个脚本加入组合区，按顺序执行一组流程。
- 鼠标键盘录制：支持录制点击、移动、按键等操作，并保存为脚本。
- 脚本编辑器：支持播放、单步执行、保存、另存为、插入常用指令。
- 找图指令：可以截取图片并生成找图脚本，用于等待按钮、图标或页面元素出现。
- 坐标拾取：快速获取屏幕坐标，并复制到脚本里使用。
- 循环执行：可设置脚本循环次数，适合重复刷图、重复检查、重复点击。
- 定时执行：支持按时、分、秒、毫秒设置触发时间，适合预约或准点操作。
- 全局快捷键：支持 F11/F12 等快捷控制，用于暂停、停止、单步等操作。
- 悬浮控制窗：录制或执行时可以通过悬浮按钮停止，减少切回主窗口的麻烦。
- 脚本文件管理：集中管理脚本目录，支持打开、刷新、编辑和绑定。
- 本地配置保存：窗口位置、按钮绑定、脚本路径等会保存到本地配置文件。

## 脚本能力

脚本文件通常是 `.txt` 或 `.ks` 文本文件，每行是一条指令。项目内置的脚本引擎支持常见自动化命令，例如：

```text
MoveTo 100,200
LeftClick 1
Delay 1000
KeyPress F1
Call "scripts/demo.txt"
MsgBox "任务完成"
For 5
    LeftClick 1
    Delay 500
Next
```

常见指令包括：

- `MoveTo`：移动鼠标到指定坐标。
- `LeftClick` / `RightClick`：左键或右键点击。
- `LeftDown` / `LeftUp`：按下或松开鼠标左键，适合拖拽类操作。
- `Delay`：等待指定毫秒数。
- `KeyPress` / `KeyDown` / `KeyUp`：键盘按键操作。
- `CopyPaste`：复制粘贴文本。
- `FindPic`：在屏幕区域内查找指定图片。
- `Call`：调用另一个脚本，方便拆分复杂流程。
- `For` / `Next`、`Do` / `Loop`：循环控制。
- `If` / `EndIf`：条件判断。
- `WaitUntil`：等待到指定时间。
- `RunApp`：启动外部程序。
- `Stop`：停止当前流程。

## 示例用法

### 阴阳师日常脚本

可以把常见流程拆成多个小脚本，例如：

- `进入活动.txt`
- `领取每日奖励.txt`
- `御魂重复挑战.txt`
- `返回庭院.txt`

然后把这些脚本分别绑定到主控台按钮，或者加入右侧组合区，一键按顺序执行。

### 养成类游戏快捷脚本

适合把固定步骤整理成脚本，例如：

```text
MoveTo 1200,680
LeftClick 1
Delay 800
MoveTo 980,620
LeftClick 1
Delay 1000
```

如果入口或按钮位置会变化，可以配合 `FindPic` 查找按钮图片，再执行后续点击。

### 抢票或预约辅助

可以使用定时执行功能，在指定时间触发脚本。例如提前打开页面后，让脚本在目标时间执行刷新、点击或确认。

使用这类功能时请特别注意：不要绕过验证码、风控、人机验证或平台限制；网络延迟、页面变化和平台规则都可能导致脚本失败。

## 运行环境

- Windows
- Python 3.10 或更高版本

## 安装依赖

```powershell
pip install -r requirements.txt
```

如果需要更稳定的找图识别，可以额外安装 OpenCV：

```powershell
pip install opencv-python
```

## 启动方式

直接运行主程序：

```powershell
python anjian20251212.py
```

也可以在 Windows 下运行批处理：

```powershell
.\anjian26.bat
```

部分游戏或需要较高权限的窗口可能无法被普通权限控制。遇到快捷键、点击或找图不生效时，可以尝试以管理员身份运行终端或程序。
